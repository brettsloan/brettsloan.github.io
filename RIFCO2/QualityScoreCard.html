Quality Score Card:

Call Open

* Did the Agent correctly disclose the Outbound Call Recording Disclosure Statement?

 * Were the proper points of customer verification completed?

Collection Core Skills

 * Did the Agent identify and discuss the Reason for Delinquency?

  Did the Agent discuss and/or challenge account history, including payment history details within the conversation?
  (IE: Broken promises, NSF's, difficult contact, abnormal RFD patterns, potentially compromised collateral or customer info)

Responses / Rebuttals

  Did the Agent use appropriate responses to all client objections?

Promise To Pay / Payment

  Did the Agent outline the total amount due & make arrangements for the full arrears balance?

* Did the Agent negotiate to obtain payment arrangements and determine what could be paid today?

  Did the Agent discuss Source of Funds and/or employment information to determine PTP validity?

  Did the Agent address/discuss the the next regular payment due?

  Did the Agent discuss the benefits and possible consequences of payment/non-payment to create urgency?

* Did the Agent recite the Mini-Miranda?
(Prior to processing a Blue Pay - Visa Debit/Master card payment)

Professionalism

  Did the Agent project courtesy and professionalism?
  (IE: Strong product and client knowledge, proper tone, pace, control, word usage, etc.)

Third Party Contact / Messages

  Were appropriate third party guidelines followed?
  (Reference, employer, DNC'd third parties)

Call Wrap Up

  Did the Agent or client recap all payment dates?

  Did the Agent or client recap all payment amounts?

  Does the agent's note and actioned statuses reflect how the call went and include all pertinent details?
  (Including the update of all demographics)

* Did the Agent disposition the call properly through LiveVox?

  Did the Agent set a proper follow-up date?

* If this criteria is missed, the agent scores zero for the entire category
